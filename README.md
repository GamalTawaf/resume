# Gamal Tawaf — Resume

Personal portfolio and resume site: about/experience, certifications, projects, a game center, and a page of lessons learned building an AI agent for the Hugging Face AI Agents Course.

**Live site:** https://gamaltawaf.github.io/resume/

## Pages

- **Home / About** — background, work history, skills, education, certifications
- **Projects** — repos, including the GAIA multi-agent built for the Hugging Face AI Agents Course
- **Games** — small games built to learn new languages and frameworks (React, Rust/WASM, Svelte)
- **Vibe Coding** — real lessons learned from building with an LLM in the loop

## Run locally

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deploy

Pushes to `main` deploy automatically via GitHub Actions.
